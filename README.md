# iloveshare
Just an webapp like dev.to. but in french language only
Users can be connected and add articles, tags, pictures. 

## Stack
- React 18
- Strapi v4

## Packages
- Chakra UI
- React Router
- Axios
- Redux toolkit
- Jest
- Nanoid
- React markdown
- React Icons
